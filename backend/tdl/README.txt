# Ledgerit TDL Installation Guide

## How to Install Ledgerit TDL in Tally

1. **Extract the ZIP file** to get `Ledgerit.tdl`

2. **Copy TDL file** to Tally installation directory:
   - Default path: `C:\Program Files (x86)\Tally.ERP 9\`
   - Or your custom Tally installation folder

3. **Load TDL in Tally**:
   - Open Tally
   - Press `F12` (Configure)
   - Go to `TDL & Add-On` â†’ `TDL Management`
   - Click `Load TDL` and select `Ledgerit.tdl`

4. **Access Ledgerit Import**:
   - From Gateway of Tally, you'll see "Ledgerit" option
   - Click on it to import XML files generated by Ledgerit

## Features

- Direct XML import from Ledgerit
- Automatic voucher creation
- Bank statement integration
- Easy-to-use interface
